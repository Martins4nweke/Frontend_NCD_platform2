# NCD Cascade Allocation Frontend

Fresh Vite + React frontend aligned with the generic NCD cascade FastAPI backend.

## Features
- Disease module browser
- Upload Excel or CSV
- Run allocation
- Compare scenarios
- Download Word report
- National policy brief
- Equity recommendations
- Province-specific briefs
- KPI cards
- Grouped summaries and top allocation rows
- Validation warnings and provenance summary

## Required Vercel environment variable
VITE_API_BASE=https://your-backend.onrender.com

## Local run
npm install
npm run dev

## Build
npm run build
